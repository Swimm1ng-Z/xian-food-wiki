# 🥟 西安美食百科

西安经典美食交互式地图 —— 52 道经典美食、189 家真实店铺，一站式浏览、收藏、投稿、评论。

👉 **[打开网站](https://swimm1ng-z.github.io/xian-food-wiki/)**

## 功能

- **🗺️ 交互地图** — Leaflet.js 驱动的西安美食地图，每家店铺独立标记，点击查看详情
- **🔍 分类筛选** — 按面食/泡馍/凉皮/小吃/饮品/汤羹等 7 大分类浏览
- **❤️ 收藏夹** — 收藏喜欢的美食，侧边栏快速筛选
- **💬 评论** — 每道美食底部 Giscus 评论区，用 GitHub 账号登录即可发言
- **📝 投稿** — 填写美食名称和地址，高德地图自动定位生成坐标，通过 GitHub Issue 提交
- **🌓 暗色模式** — Light/Dark 主题切换，地图同步

## 技术栈

纯静态单页应用，零框架依赖：

- **Leaflet.js** — 开源地图库
- **高德地图 API** — 地址搜索与地理编码
- **Giscus** — 基于 GitHub Discussions 的评论系统
- **GitHub Pages** — 免费托管
- 52 道美食、189 家真实店铺，全部由人工验证定位

## 投稿新美食

1. 打开网站 → 点击「投稿新美食」
2. 填写美食名称、分类、店铺名称
3. 输入店铺地址，高德地图自动补全并生成坐标
4. 提交到 GitHub Issue，审核后加入数据库
