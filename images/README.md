# 美食图片文件夹

## 使用方法

将每道美食的图片放入此文件夹，命名为 `{美食ID}.jpg`。

例如：
- `1.jpg` → Biangbiang面（ID=1）
- `9.jpg` → 羊肉泡馍（ID=9）
- `15.jpg` → 腊汁肉夹馍（ID=15）
- `36.jpg` → 冰峰汽水（ID=36）

## 图片来源建议

- 百度图片搜索：https://image.baidu.com/
- 大众点评商家图
- 自己拍摄的美食照片

## 图片要求

- 推荐尺寸：800×500 像素
- 格式：JPG 或 WebP
- 建议每张不超过 200KB（GitHub Pages 加载更快）
